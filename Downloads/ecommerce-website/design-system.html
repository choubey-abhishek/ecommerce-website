Kopal Seth Studio

A premium, editorial design language for a contemporary ceramics house — warm neutrals, an earthy accent palette, Fraunces & Inter, and a quiet, gallery-like restraint. This is a living reference; every value here is a token.

Primitive ramps map to semantic roles. Components reference roles, never raw hex. Every neutral is warm; every semantic is muted and earthy.

Spacing & grid

A 4px base scale. Section rhythm is 96 → 128px; gutters 24 → 40px. Containers: narrow 768 · default 1152 · wide 1280.

Scale (4px base)

Semantic spacing

Card padding — 24px mobile → 32px desktop
Section rhythm — py 96px → 128px
Page gutter — 24px → 40px
Stack gaps — tight 8–12 · default 16–24
Product grid — cols 2 · lg 3 · xl 4, gap-x 24 / gap-y 48 (standardized site-wide)

Border radius

Soft and editorial. Buttons, pills, and search inputs are fully round; cards use large radii.

Shadow

Wide, low-opacity ink shadows — never harsh.

Motion

One easing curve, short durations, full reduced-motion support.

Easing
ease-studio — cubic-bezier(0.22, 1, 0.36, 1) · primary
ease-standard — cubic-bezier(0.4, 0, 0.2, 1) · utility

Duration
fast 150ms · base 250ms · moderate 300ms · slow 500ms · slower 800ms (reveals)
Principles. Enter with fadeUp / scaleIn; exit faster than enter. Scroll reveals use whileInView with once: true and a no-JS fallback so content is never permanently hidden. Under prefers-reduced-motion, drop parallax, transforms, autoplay, and infinite loops — keep opacity fades ≤150ms.
